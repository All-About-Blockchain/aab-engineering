📊 Free Endpoints (No API Key)
GET /health
Health check
GET /v1/chains
List supported blockchain networks
GET /v1/rates
Yield rates across protocols
GET /v1/bridge/config
Bridge configurations (Stargate, LayerZero, Wormhole, Axelar)
GET /v1/bridge/chains
Supported bridge chains
GET /v1/swap/tokens
Swap token list
GET /v1/staking/rates
Staking APYs (stETH, rETH, jitoSOL, mSOL)
🔐 Protected Endpoints (API Key Required)
POST /v1/bridge/quote
Get bridge quote
POST /v1/swap/quote
Get swap quote (Jupiter/1Inch)
POST /v1/staking/deposit
Get staking deposit transaction
POST /v1/staking/withdraw
Get staking withdraw transaction
POST /v1/wallet/create
Create MPC wallet (TurnKey)
POST /v1/wallet/:walletId/sign
Sign transaction
GET /v1/moonpay/buy-url
Get MoonPay widget URL for buying crypto
GET /v1/moonpay/quote
Get MoonPay buy quote

Example: curl -H "X-API-Key: aab_cron_key_2026" https://aab.engineering/v1/wallet/create
