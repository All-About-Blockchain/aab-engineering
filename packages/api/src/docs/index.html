📊 Core
GET /v1/rates
Returns all available yield rates across supported protocols
Query protocol (optional), chain (optional)
GET /v1/rates/etherfi
EtherFi specific rates (weETH, weETHs) - cached 1hr
Response { weETH: { apy, tvl, supply }, weETHs: { apy, tvl, supply }, ethPrice }
GET /v1/rates/protocols
List all supported protocols
Response [{ name, chain, apy, tvl, risk }]
⚡ Execute
POST /v1/execute AUTH
Execute a yield transaction (stake, unstake, swap)
Headers X-API-Key: your-api-key
Body { action, protocol, amount, token, fromAddress, toAddress }
GET /v1/execute/status/:txId
Check transaction status by tx hash
Response { status, confirmations, blockNumber, gasUsed }
GET /v1/execute/simulate
Simulate a transaction without executing
Query action, protocol, amount, token
📈 Analytics
GET /v1/analytics
Usage statistics and volume
Response { requests, volumeUsd, uniqueUsers, avgResponseTime }
GET /v1/analytics/history
Historical analytics data
Query period (1d, 7d, 30d)
